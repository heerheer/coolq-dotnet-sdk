    HuajiTech.QQ.Events 命名空间

    类

    AnonymousMessageReceivedEventArgs

    为 AnonymousMessageReceived 事件提供数据。

    EntranceInvitedEventArgs

    为 EntranceInvited 事件提供数据。

    EntranceRequestedEventArgs

    为 EntranceRequested 事件提供数据。

    FileUploadedEventArgs

    为 FileUploaded 事件提供数据。

    FriendedEventArgs

    为 Friended 事件提供数据。

    FriendingEventArgs

    为 Friending 事件提供数据。

    GroupEventArgs

    为群事件提供数据。

    GroupMuteEventArgs

    为 GroupMuted 和 GroupUnmuted 事件提供数据。

    MemberMutedEventArgs

    为 MemberMuted 事件提供数据。

    MessageReceivedEventArgs

    为 MessageReceived 事件提供数据。

    RoutedEventArgs

    表示路由事件的数据，并为所有事件数据提供基类。

    TimedEventArgs

    为可以提供引发时间的事件数据提供基类。

    接口

    定义群管理员调整事件。

    IBotEventSource

    定义机器人事件。

    ICurrentUserEventSource

    IFriendshipEventSource

    定义好友事件。

    IGroupEventSource

    定义群事件。

    IGroupMuteEventSource

    定义群禁言事件。

    IMemberMuteEventSource

    定义成员禁言事件。

    IMembershipEventSource

    定义群成员事件。

    IMessageEventSource

    定义消息事件。

    IMuteEventSource

    定义禁言事件。

    INotifyAdministratorAdded

    定义管理员添加事件。

    INotifyAdministratorRemoved

    定义管理员移除事件。

    INotifyAnonymousMessageReceived

    定义匿名消息接收事件。

    INotifyAppDisabling

    定义应用禁用事件。

    定义应用启用事件。

    INotifyBotStarted

    INotifyBotStopping

    定义机器人停止事件。

    INotifyEntranceInvited

    定义入群邀请事件。

    INotifyEntranceRequested

    定义入群请求事件。

    INotifyFileUploaded

    定义文件上传事件。

    INotifyFriended

    定义好友已添加事件。

    INotifyFriending

    定义好友正在添加事件。

    INotifyGroupMuted

    定义群禁言事件。

    INotifyGroupUnmuted

    定义群解除禁言事件。

    INotifyMemberJoined

    定义成员加入事件。

    INotifyMemberLeft

    定义成员离开事件。

    INotifyMemberMuted

    定义成员禁言事件。

    INotifyMemberUnmuted

    定义成员解除禁言事件。

    INotifyMessageReceived
