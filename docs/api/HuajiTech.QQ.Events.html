    HuajiTech.QQ.Events 命名空间

    类

    AnonymousMessageReceivedEventArgs

    为 AnonymousMessageReceived 事件提供数据。

    ContactAddedEventArgs

    为 ContactAdded 事件提供数据。

    ContactRequestedEventArgs

    为 ContactRequested 事件提供数据。

    EntranceInvitedEventArgs

    为 EntranceInvited 事件提供数据。

    EntranceRequestedEventArgs

    为 EntranceRequested 事件提供数据。

    FileUploadedEventArgs

    为 FileUploaded 事件提供数据。

    GroupEventArgs

    为群事件提供数据。

    GroupMemberEventArgs

    为 MemberJoined 和 MemberLeft 事件提供数据。

    GroupMuteEventArgs

    为 GroupMuted 和 GroupUnmuted 事件提供数据。

    MemberMutedEventArgs

    为 MemberMuted 事件提供数据。

    MessageReceivedEventArgs

    为 MessageReceived 事件提供数据。

    RoutedEventArgs

    表示路由事件的数据，并为所有酷Q事件数据提供基类。

    TimedEventArgs

    为可以提供引发时间的事件数据提供基类。

    接口

    IBotEventSource

    定义机器人事件。

    IContactEventSource

    定义联系人事件。

    IEntranceInviteEventSource

    定义入群邀请事件。

    IEntranceRequestEventSource

    定义入群请求事件。

    IFileEventSource

    定义文件事件。

    IGroupAdministratorEventSource

    定义群管理员事件。

    IGroupMemberEventSource

    定义群成员事件。

    IGroupMuteEventSource

    定义群禁言事件。

    IMemberMuteEventSource

    定义成员禁言事件。

    IMessageEventSource

    定义消息事件。
