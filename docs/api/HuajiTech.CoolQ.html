    HuajiTech.CoolQ 命名空间

    类

    ApiException

    在调用 API 发生错误时引发的异常。

    AppIdAttribute

    指定当前程序集的 AppID。 此类不能被继承。

    CurrentPluginContext

    提供操作 Current 的静态方法。

    CustomTitle

    表示自定义头衔。

    Extensions

    定义扩展方法。

    File

    表示文件。

    Logger

    提供记录日志的方法。

    Plugin

    用作插件的基类，并提供操作 PluginContext 的实例方法。 此类为抽象类。

    PluginContext

    表示插件上下文。 此类为抽象类。

    PluginContextExtensions

    定义用于操作 PluginContext 的扩展方法。

    PluginLoadStageAttribute

    指定插件加载阶段。 此类不能被继承。

    Status

    表示状态。

    接口

    IAliased

    定义有别名的实例。

    IAnonymousMember

    定义匿名成员。

    IBot

    定义机器人。

    IChattable

    定义可聊天的实例。

    ICurrentUser

    定义当前用户。

    IDisplayable

    定义可显示的实例。

    IEntranceRequest

    定义加入请求。

    IFriend

    定义好友。

    IFriendshipRequest

    定义好友请求。

    IGroup

    定义群。

    ILoader

    定义加载器。

    ILogger

    定义记录日志的方法。

    IMember

    定义成员。

    IMessage

    定义消息。

    IMuteable

    定义可禁言实例。

    IPacker

    定义打包器。

    IPlugin

    用于标记插件类。

    IRefreshable

    定义可刷新的实例。

    IRequest

    定义请求。

    IRequestable

    定义可请求的实例。

    ISendee

    定义可发送消息的实例。

    ITimedMuteable

    定义可以指定时间的可禁言实例。

    IUser

    定义用户。

    枚举

    AppLifecycle

    指定应用生命周期。

    Gender

    指定性别。

    LogLevel

    指定日志级别。

    MemberRole

    指定成员角色。

    StatusColor

    指定状态颜色。
