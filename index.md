# HuajiTech.CoolQ

用于酷Q应用的 .NET SDK。

**项目目前处于测试阶段，不建议用于生产环境。**

## 特性

- 通过 StdCall 导出。
- 以 [NuGet 包](https://www.nuget.org/packages/HuajiTech.CoolQ/)发布。
- **不使用易语言的开发模式。**
